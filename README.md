# Flights Booking WebApi

Đồ án phục vụ môn học "Phát triển phần mềm hướng dịch vụ".

### Các chức năng

Cung cấp các api về:

1. **Chuyến bay**
2. **Hóa đơn**
3. **Khách hàng**
4. **Máy bay**
5. **Nhân viên**
6. **Tài khoản**
7. **Vé**

### Các công nghệ sử dụng

- MS SQL Server.
- Asp.net WebApi, Entity Framework Code First.
- Khác:
  - Mô hình 3 layers (Data, Service, Presentation).
  - IoC (Autofac).
  - Swagger.
